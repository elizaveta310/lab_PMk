# Теоретичні відомості

**PSoC Creator** — це інтегроване середовище розробки (IDE), яке використовується для програмування мікроконтролерів сімейства **PSoC** від Cypress.

Основні можливості PSoC Creator:

- створення та керування проєктами
- конфігурування апаратних компонентів
- автоматична генерація коду
- компіляція та відлагодження програм
- використання готових прикладів (Code Examples)

У даній роботі було розглянуто процес створення простого проєкту керування світлодіодом за допомогою кнопки.

# Програмна реалізація

Основний код керування світлодіодом:

```c
#include "project.h"

int main(void)
{
    CyGlobalIntEnable; /* Enable global interrupts. */

    for(;;)
    {
        if(Button_Read() == 1)
        {
            LED_Write(0);
        }
        else
        {
            LED_Write(1);
        }
    }
}
```

Принцип роботи програми:

- якщо **кнопка не натиснута** — світлодіод увімкнений
- якщо **кнопка натиснута** — світлодіод вимкнений

## Необхідне програмне забезпечення

Для виконання роботи необхідно:

- **PSoC Creator 4.4**
- плата **CY8CKIT-042 PSoC 4 Pioneer Kit**
- USB підключення до комп’ютера

---

## Кроки запуску

1. Відкрити **PSoC Creator**
2. Створити новий проєкт
3. Обрати мікроконтролер **CY8C4245AXI-483**
4. Додати компоненти **Digital Input Pin** та **Digital Output Pin**
5. Налаштувати піни
6. Скомпілювати проєкт (**Build**)
7. Завантажити програму на плату (**Program**)
